# UnSpookify - Face Your Fears, Find Your Calm

A Cognitive Behavioral Therapy (CBT) web application designed to help users face and overcome their fears through exposure therapy and guided exercises.

## Features

- **Exposure Therapy**: Interactive tools for gradual exposure to fears
- **Journaling**: Track progress and reflections
- **Mood Tracking**: Monitor emotional states over time
- **AI Therapist**: Get guided support through challenging moments
- **Progress Dashboard**: Visualize your improvement journey

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/unspookify.git
   cd unspookify
   ```

2. Install dependencies for both root and client:
   ```bash
   # Install root dependencies
   npm install
   
   # Install client dependencies
   cd client
   npm install
   cd ..
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory with the required environment variables.

### Running the Application

1. Start the development server:
   ```bash
   # Start both client and server with concurrent
   npm run dev
   
   # Or start them separately
   # In one terminal (root):
   npm start
   # In another terminal (client):
   cd client && npm start
   ```

2. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
