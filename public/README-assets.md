# Relaxation Features Assets

This directory contains assets used by the relaxation features of the application.

## Audio Files

- `sounds/breathing-guide.mp3`: A silent audio file used for the breathing exercise timer.

## Replacing Placeholder Images

To enhance the user experience, replace these placeholder images with actual illustrations or photos of each muscle group. Each image should clearly show the muscle group being targeted in the exercise.

## Audio Guidance

The `breathing-guide.mp3` is currently a silent file. For a better user experience, consider recording or obtaining a guided breathing audio track that matches the 4-4-4-4 breathing pattern.
